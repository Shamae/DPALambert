How to get this running:
    - install .Net Core (Visual Studio Code: https://www.microsoft.com/net/core#windowscmd)
    - install C# (Visual Studio Code: https://marketplace.visualstudio.com/items?itemName=ms-vscode.csharp)

    - execute "dotnet restore" and "dotnet run"
